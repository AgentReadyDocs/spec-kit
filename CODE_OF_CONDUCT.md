# Code of Conduct

This project follows the Contributor Covenant principles.

- Be respectful and constructive.
- No harassment, hate speech, or discriminatory behavior.
- Assume good intent and focus on actionable feedback.

## Enforcement

- For non-sensitive reports: open an issue in the relevant repository.
- For sensitive reports: do not include personal details; open a minimal issue requesting a private contact path, or use GitHub's reporting tools.
